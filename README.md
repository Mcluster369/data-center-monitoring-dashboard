# Data Center Monitoring Dashboard

## Project Overview

This project is a Python dashboard that simulates and visualizes data center server health, temperature, power usage, and network activity. It is designed as a portfolio project for entry-level data center, IT support, and infrastructure monitoring roles.

The goal is to demonstrate how operational data can be monitored, reviewed, and communicated clearly using Python-based analysis and visualization.
